Frontend User Management - Usage Instructions

1. Installation:

   a. Clone the repository:
      ```
      git clone https://github.com/yourusername/frontend_user_management.git
      ```

   b. Navigate to the project directory:
      ```
      cd frontend_user_management
      ```

   c. Install the required dependencies:
      ```
      npm install
      ```

2. Usage:

   a. Start the server in development mode:
      ```
      npm run devstart
      ```

   b. Open your web browser and go to http://localhost:5000 to access the frontend.

   c. Explore the API at http://localhost:5000/api.

3. Project Structure:

   - frontend.server.js: The main server file.
   - index.html: The HTML file served by the server.
   - node_modules/: Dependencies required for the project.
   - package.json: Project configuration and dependencies.
   - package-lock.json: Auto-generated file locking dependency versions.
   - scripts/: Directory for project scripts or other code.

4. Scripts:

   - `npm start`: Start the server in production mode.
   - `npm run devstart`: Start the server in development mode with auto-reloading (using Nodemon).
